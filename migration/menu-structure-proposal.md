# Proposed Menu Structure

Date: 2026-04-19
Status: proposal only

## Recommendation

Do **not** recreate the current Weebly menu one-to-one.

Reason:
- the current site effectively exposes too many deep links directly in navigation
- it is hard to scan
- it is not a good guide for readers
- a public static site should help people find the right section first, then the right page

Instead, keep the content and URLs, but simplify the primary navigation into a small number of clear top-level sections.

## Proposed primary navigation

- Home
- Club
- Fleets
- Racing
- News & Archives
- Photos
- Useful Links

### News & Archives

### Photos
- Photos landing page
- gallery and slideshow pages grouped by event/year

### Useful Links
- Useful Links

## Homepage menu behavior

For the homepage, I recommend:
- a clean top navigation bar with the section names above
- no giant mega-menu yet
- section landing pages should do the deeper routing work

## Why this is the best compromise

This keeps the site:
- easier to read
- easier to maintain
- more member-friendly
- less cluttered than the current menu

while still preserving the actual content pages underneath.
